Regular Expressions

Basics

Regular expressions are patterns
used to match character combinations in strings

Creation

Flags

Regular expressions have 8 optional flags.
Flags can be used separately or together in any order.

g i s m u v y d

Flag g - global search

Flag i - case-insensitive search

Flag s - enables "dotAll" mode

Flag m - multiline mode

Character classes

are special notation that matches any symbol from a certain set

\d \D \s \S \w \W .

\d - any single digit

\D - any character except digits

\s - space characters

\S - any character except space characters

\w - 'wordly' characters

letters of Latin alphabet, digits or '_'

\W - any character except 'wordly' characters

A dot . - any character except a newline '\n'

Anchors

are two special characters: ^ and $

^ - matches at the beginning of the string

$ - matches at the end of the string

Special characters

have special meaning in Regular Expressions

[ ] { } ( ) \ ^ $ . | ? * +

Sets and ranges [ ]

search any characters among written inside

Quantifiers { } + * ?

search exactly numbers of characters

{n} specify how many characters we need

+ - one or more

? - zero or one

* - zero or more

Capturing groups ( )

use quantifier after the parentheses to apply to everything inside

Alternation |

is simple 'or'
